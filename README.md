## How to use?

### Installation:

Enter and run the following sentence in your console:

```html
npm install
```

### Run the project:

Enter and run the following sentence in your console:

```html
npm start
```

to enter developer mode.

Finally, open [http://localhost:4321](http://localhost:4321) to see it in your default browser.
